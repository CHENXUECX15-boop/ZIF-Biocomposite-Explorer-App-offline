# Guideline for the Offline Version of the ZIF Biocomposite Explorer App

The offline version of the ZIF Biocomposite Explorer App is designed for browsing and filtering ZIF biocomposite synthesis data. The software displays the samples in a stacked TD format according to TC and synthesis conditions, and links each sample to its corresponding phase, PXRD pattern, ATR-IR spectrum, EE%, LC%. The data can be viewed, filtered, and compared locally without an internet connection. To use the software, the compressed file provided in the Supporting Information should first be extracted while keeping the original folder structure unchanged. The offline app can then be launched by opening the main file, index.html, in a web browser.

The interface consists of three main sections: a control panel on the left, stacked ternary diagrams in the center, and a sample information panel on the right. The left control panel allows users to select the dataset, concentration, visualization, and filters. The central panel displays TDs at different TCs, where the three vertices correspond to M%, L%, and BSA%. The stacked layers represent different TCs, including 12.5, 25, 50, 75, and 100~mg~mL$^{-1}$. The right panel displays detailed compositional, phase, properties, and characterization information for the selected sample.

In the left control panel, the Dataset section allows users to switch between the WW and EW washing conditions. In the Concentration section, users can display all TCs or select a specific TC. In the Visualization section, different data-display modes can be selected. In Phase mode, colors indicate different phases, including amorphous (grey), ZIF-C (light blue), sod (dark blue), dia (red), U12 (green), U13 (yellow), and mixed phases (purple). In EE% or LC% mode, the sample points are colored according to the corresponding EE% or LC%, with red indicating higher values and white indicating lower values. The Filters section allows users to define minimum and maximum values for M%, L%, BSA%, EE%, and LC%. The Reset Filters button restores the default view. The Phase filter enables the display of one selected phase category or all phase categories. In the navigation section, the Rotate and Pan modes allow users to adjust the three-dimensional view, while the Reset View button restores the default viewing angle. The current visualization can be exported as a PNG image using the Download PNG button.

When a sample point in the ternary diagram is selected, the right information panel displays the corresponding washing condition, sample number, TC, phase, M%, L%, BSA%, L/M ratio, EE%, and LC%. The table below summarizes the phase, EE%, and LC% for the same synthesis coordinate across different TCs, allowing concentration-dependent changes to be compared directly. The PXRD patterns section shows the PXRD data from triplicate syntheses, which can be used to assess phase composition and reproducibility. The ATR-IR spectrum section displays the corresponding IR spectrum, supporting the analysis of ZIF framework features and protein-related signals. Both the PXRD and ATR-IR panels provide independent controls, including zoom in, zoom out, pan, reset, and export options for downloading the displayed data or figure as CSV and PNG files.

AI-assisted software generation: Custom scripts for generating TDs for data visualization in the phase-ternary-site.zip package were developed with the assistance of Codex-GPT-5.5 (accessed May 2026). All AI-generated code was reviewed, tested, and revised by the authors, who confirmed accuracy and reproducibility of results.
